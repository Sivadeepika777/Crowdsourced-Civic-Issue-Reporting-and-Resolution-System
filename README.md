AI Civic Guardian

AI-Powered Civic Issue Reporting and Resolution System built using Flutter, FastAPI, MongoDB, TensorFlow, and OpenCV.

Problem Statement

Citizens often face difficulties in reporting civic issues such as potholes, garbage accumulation, water leakage, and damaged streetlights. Existing complaint systems are slow, lack transparency, and provide limited tracking facilities.

Objective

The objective of this project is to provide a smart platform that enables citizens to report civic issues easily and helps authorities resolve them efficiently using AI-powered automation.

Features

- User Authentication
- Civic Issue Reporting
- Image Upload
- GPS Location Tracking
- AI Issue Classification
- Duplicate Complaint Detection
- Complaint Status Tracking
- Notifications
- Dashboard Analytics
- Admin Management
